This project recreates a given PDF document into an accurately formatted MS Word (.docx) file using Python.
The main goal is to replicate the PDF layout precisely, including headings, spacing, alignment, line breaks, and overall structure.
The solution is built using python-docx and includes a Streamlit web interface for easy demonstration and deployment.
Convert a structured PDF document into a Microsoft Word document while:
Preserving layout and formatting
Maintaining readability
Ensuring accurate alignment and spacing
Producing a professional, editable Word file

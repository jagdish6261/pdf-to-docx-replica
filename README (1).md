
# PDF to MS Word Replica Project

## Overview
This project recreates a given PDF document into an **exactly structured MS Word (.docx) file** using Python.
The goal is to match layout, spacing, headings, alignment, and readability precisely.

## Tech Stack
- Python 3.x
- python-docx
- pdfplumber (optional for automation)
- Streamlit (for deployment)

## Features
- Exact replication of PDF layout
- Clean and readable Word document
- Modular and maintainable code
- Ready for deployment

## Project Structure
```
pdf-to-docx-replica/
├── input/
│   └── mediation_form.pdf
├── output/
│   └── Mediation_Application_Form.docx
├── main.py
├── app.py
├── requirements.txt
└── README.md
```

## How to Run Locally
```bash
pip install -r requirements.txt
python main.py
```
